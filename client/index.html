This is a temporary page. Our application now uses Next.js which is running on a separate port.

How to Access TinyPaws Next.js App in Replit:

  1. Look for the Webview panel in Replit (usually on the right side)
  2. Click on the URL bar in the Webview panel
  3. Change the port from 5000 to 3000 in the URL
  4. Press Enter to navigate to the Next.js app

Note: If you don't see the Webview panel, click on the "Webview" button in Replit's top navigation bar.

If you're seeing this page, it means the Express API server is running on port 5000, but you need to access the application through the Next.js server on port 3000.
